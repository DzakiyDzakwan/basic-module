Display Inline

Display inline akan membuat sebuah elemen bersifat inline.

Yang artinya elemen tersebut akan di posisikan di sebelah kanan dari elemen sebelumnya atau tersusun secara horizontal.

Walaupun elemen tersebut merupakan elemen block level.

Display Block

Display block akan membuat sebuah elemen bersifat block. Display block akan membuat elemen tersusun secara vertikal.

Display Inline Block

Display inline block akan membuat elemen tersusun secara horizontal seperti elemen inline.

Namun elemen tersebut akan memiliki fitur block yaitu bisa di atur dimensi nya.
